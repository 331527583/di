Introduction & Motivation

DI is C++03/C++11 header only library providing compile time, macro free constructor dependency injection functionality.

"Don't call us, we'll call you", Hollywood principle

  • for using Dependency Injection
  • for using Dependency Injection Framework 

#include <boost/di.hpp>

class app {
public:
    app(const std::shared_ptr<int>&
      , boost::shared_ptr<int>
      , std::unique_ptr<int>
      , double
      , int
      , const std::string&)
    { }
};

int main() {
    auto injector = boost::di::make_injector();

    app = injector.create<app>();

    //...

    return 0;
}

include di headers

Define class as usual

make injector

create app

use app
