Modules

Modules allow to split the configurations into smaller injectors. Module is a class with configure method returning injector type. 

class module {
public:
    auto configure() const {
        return di::make_injector(
            ...
        );
    }
}

Activation of Module is done by passing it into injector.

di::make_injector

di::injector 

auto injector = di::make_injector(module(), ...);


using injector = di::injector<module, ...>;

Tip

Modules/Concepts/Types might be mixed whilst creating an injector, order is not important
