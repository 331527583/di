BOOST_DI_INJECT_TRAITS

BOOST_DI_INJECT_TRAITS is a macro definition used to define constructor traits.

Definition

Preprocessed 

#define BOOST_DI_INJECT_TRAITS(...) \
    static void BOOST_DI_INJECTOR(__VA_ARGS__)


BOOST_DI_INJECT_TRAITS(int, double, float);

    void boost_di_injector__(int, double, float);

Boost.DI defines ctor_traits in order to handle cases when constructor can't be chosen using BOOST_DI_INJECT (useful for third party libraries). 

namespace boost { namespace di {

template<>
struct ctor_traits<example> {
    BOOST_DI_INJECT_TRAITS(int, double, float);
};

}} // boost::di

Note

BOOST_DI_INJECT_TRAITS might be also defined within the class, useful in case of named parameters
