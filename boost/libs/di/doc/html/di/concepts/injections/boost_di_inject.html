BOOST_DI_INJECT

Description

BOOST_DI_INJECT is a macro definition used to explicitly say Boost.DI which constructor should be used. When class has more than one constructor Boost.DI will by default choose the one with the longest parameter list. In case of constructors ambiguity Boost.DI is not able to choose the best one. Then BOOST_DI_INJECT become handy to point which constructor should be used.

Synopsis 
#define BOOST_DI_INJECT(type, ...)
    static void BOOST_DI_INJECTOR(__VA_ARGS__);
    type(__VA_ARGS__)

BOOS_DI_INJECT(explicit example, int = 0, double = 0.0);

Preprocessed type: 

void boost_di_injector__(int = 0, double = 0.0);
explicit example(int = 0, double = 0.0);
Header 
#include <boost/di.hpp>
Example 
class example {
public:
    example(int);
    BOOS_DI_INJECT(example, int, double); // will be chosen by Boost.DI
    example(int, double, float);
};

assert(di::make_injector().create<example>());
Note

In C++03 additional brackets are needed: BOOST_DI_INJECT(example, (int, double)).

Tip

Requirement of brackets in BOOST_DI_INJECT might be switched with BOOST_DI_CFG_INJECT_VA_ARGS and BOOST_DI_CFG_INJECT_BRACKET.
