deduce
Description

Allows to bind other bindings to deduce scope scope. Interface type will be deduced, if not given. Scoped will be deduced accordingly to constructor parameter type. See Deduce Scope for further information.

Synopsis 
template<typename... TArgs>
struct deduce
    : di::scope<scopes::deduce>::bind<TArgs...>
{ };
