Overview

Dependencies

  • C++14 compliant compiler with STL (memory, type_traits headers) / Boost is not required

Supported Compilers

  • Clang-3.4+
  • GCC-5.1+
  • Microsoft Visual Studio 2015*+

* - limited support

Installation

To get started the only file you need is di.hpp:

https://raw.githubusercontent.com/krzysztof-jusiak/di/cpp14/include/boost/di.hpp 

// main.cpp
#include "di.hpp"
int main() { }

$CXX -std=c++1y -I. main.cpp

To get and test Boost.DI library: 

git clone --recursive https://github.com/krzysztof-jusiak/di.git
cd di/build
./bootstrap.sh
./b2 -j4 ../test ../example
