make_injector

Injector might be created using di::make_injector or di::injector type it self. Preferable option is to use di::make_injector because it allows to bind dynamic values.

di::make_injector

di::injector 

auto injector = di::make_injector();


using injector_t = di::injector<>;
auto injector = injector_t();


auto injector = di::make_injector(
    di::deduce<implementation>()
);


using injector = di::injector<implementation>;
