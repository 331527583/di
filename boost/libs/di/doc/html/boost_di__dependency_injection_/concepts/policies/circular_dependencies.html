circular_dependencies

Description

Verifies constructor parameters overlapping.

Synopsis 
class circular_dependencies {
public:
    template<typename TDependency>
    void assert_policy() const;
};

Table 8. Parameters

Parameter

Requirement

Description

TDependency

Dependency Model type

dependency to be verified


Expression Semantics 
circular_dependencies();

Result: BOOST_MPL_ASSERT with CIRCULAR_DEPENDENCIES_ARE_NOT_ALLOWED or clean compilation

Header 
#include <boost/di.hpp>
Namespace 
boost::di::policies
Example 
di::make_injector().create<int>(di::policies::circular_dependencies()); // compile ok

struct cd2;
struct cd1 { cd1(cd2*); };
struct cd2 { cd2(cd1*); };

di::make_injector().create<cd1>(di::policies::circular_dependencies()); // compile error
Note

Requires BOOST_DI_CFG_EXAMINE_CALL_STACK to be defined.
