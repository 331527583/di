BOOST_DI_INJECT
Description

BOOST_DI_INJECT is a macro definition used to explicitly say Boost.DI which constructor should be used as well as in order to annotate types - see Annotations for further reding. When class has more than one constructor Boost.DI will by default choose the one with the longest parameter list. In case of constructors ambiguity Boost.DI is not able to choose the best one. Then BOOST_DI_INJECT become handy to point which constructor should be used.

Example

Direct constructor injection with ambiguous constructors 

struct c {
    c(double d, int a) : a(a), d(d) { }
    BOOST_DI_INJECT(c, int a, double d)
        : a(a), d(d) { }

    int a = 0;
    double d = 0.0;
};

auto injector = di::make_injector(
    di::bind<int>.to(42)
  , di::bind<double>.to(87.0)
);
                                                                                   .

auto object = injector.create<c>();
assert(42 == object.a);
assert(87.0 == object.d);
                                                                                                           .
Note

BOOST_DI_INJECT is limited by BOOST_DI_CFG_CTOR_LIMIT_SIZE, which by default is set to 10 and might be between 0 and 10. In order to support more constructor parameters BOOST_DI_INJECT_TRAITS_NO_LIMITS has to be used.

Example

Direct constructor injection with ambiguous constructors 

struct c {
    BOOST_DI_INJECT_TRAITS(int, double);
    c(double d, int a) : a(a), d(d) { }
    c(int a, double d) : a(a), d(d) { }

    int a = 0;
    double d = 0.0;
};

auto injector = di::make_injector(
    di::bind<int>.to(42)
  , di::bind<double>.to(87.0)
);
                                                                                   .

auto object = injector.create<c>();
assert(42 == object.a);
assert(87.0 == object.d);
                                                                                                           .

Direct constructor injection with selected constructor 

struct example {
    BOOST_DI_INJECT_TRAITS(int);
    example(int) : i(i) { }
    example(int, double) : i(i), d(d) { }

    int i = 0;
    double d = 0.0;
};
                                                                                   .

auto object = di::make_injector().create<example>();
assert(0 == object.i);
                                                                                                           .

Direct constructor injection with default value 

class example {
public:
    BOOST_DI_INJECT_TRAITS(int);
    explicit example(int i, double d = 42.0)
        : i(i) , d(d)
    { }

    int i = 0;
    double d = 0.0;
};
                                                                                   .

auto object = di::make_injector().create<example>();
assert(0 == object.i);
assert(42.0 == object.d)
                                                                                                           .

BOOST_DI_INJECT_TRAITS_NO_LIMITS
Description

BOOST_DI_INJECT_TRAITS_NO_LIMITS is similar to BOOST_DI_INJECT_TRAITS, but it's not limited by BOOST_DI_CFG_CTOR_LIMIT_SIZE and doesn't support Annotations. Useful for generated constructors.
