Description

Allows to bind interface to implementation and associate value with dependency.

Synopsis 
template<class TExpected, class TGiven = TExpected> requires boundable<TExpected, TGiven>
dependency<di::deduce, TExpected, TGiven> bind{};

template<class T1, class T2, class... Ts>
using any_of;

struct override { }; // use to override bindings with `di::bind<...> [di::override]`
Warning

MSVC requires to use with brackets when using di::bind<...>(), due to lack of variable templates.

Table 8. Parameters

Parameter

Requirement

Description

None

Interface type

None

None

Expression Semantics 
di::bind<interface, implementation>
di::bind<di::any_of<interface1, interface2>, implementation>

Return type: dependency

Header 
#include <boost/di.hpp>
Examples

Bind interface to implementation

Test 

auto injector = di::make_injector(
    di::bind<i1, impl1>
);

auto object = injector.create<unique_ptr<i1>>();
assert(dynamic_cast<impl1*>(object.get()));

Bind interface to implementation / override

Test 

auto injector = di::make_injector(
    di::bind<i1, impl1_1>
  , di::bind<i1, impl1_2> [di::override]
);

auto object = injector.create<unique_ptr<i1>>();
assert(dynamic_cast<impl1_2*>(object.get())); // binding overridden

Bind different interfaces to one

Test 

auto injector = di::make_injector(
    di::bind<bind<i1, i2>, impl>
);

auto object1 = injector.create<shared_ptr<i1>>();
auto object2 = injector.create<shared_ptr<i2>>();
assert(dynamic_cast<impl*>(object1.get()));
assert(dynamic_cast<impl*>(object2.get()));

Bind type to compile time value

Test 

template<int N> using int_ = integral_constant<int, N>;

auto injector = di::make_injector(
    di::bind<int, int_<42>>
);

assert(42 == injector.create<int>());

Bind type to value (see external scope)

Test 

auto injector = di::make_injector(
    di::bind<int>.to(42)
);

assert(42 == injector.create<int>());
