Description

Scopes are responsible for creating and maintaining life time of dependencies. If no scope will be given whilst binding deduce scope will be assumed. Boost.DI favor 6 types of scopes.

  • deduce scope (default)
  • unique scope
  • shared scope
  • singleton scope
  • session scope
  • external scope

Scopes use wrappers to return convertible objects. Wrappers are able to convert object to requested types accurately to their definition. There are 3 basic types of wrappers.

Wrapper Model 
template<class T>
class wrapper_model {
public:
    template<class TValueType>
    wrapper_model(const TValueType&) noexcept;

    operator T()() const noexcept;

Examples 
struct custom_scope {
    template<class TExpected, class>
    struct scope {
        template<class T, class TProvider>
        auto create(const TProvider& provider) {
            return shared_ptr<TExpected>{provider.get()};
        }
    };
};

auto injector = di::make_injector(
    di::bind<i1, impl1>.in(custom_scope{})
);

assert(injector.create<shared_ptr<i1>>()
       !=
       injector.create<shared_ptr<i1>>()
);
